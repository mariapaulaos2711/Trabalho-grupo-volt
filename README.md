# Objetivo:

Criamos um projeto chamado VesteEdu que foi desenvolvido para ajudar escolas no monitoramento do uso correto da farda escolar. A ideia surgiu a partir da observação de que muitos alunos não utilizam o uniforme adequadamente, causando desorganização e dificuldades na identificação dentro do ambiente escolar.

O projeto propõe um aplicativo capaz de registrar ocorrências, enviar notificações para responsáveis e auxiliar a coordenação escolar através de relatórios e acompanhamento diário.

# Resumo do objetivo:

Nosso objetivo é promover mais organização, segurança e responsabilidade na escola utilizando a tecnologia de forma prática e educativa.

# Requisitos coletados:

---

## RF — Requisitos Funcionais

RF01 — O sistema deve registrar alunos.

RF02 — O sistema deve registrar o uso da farda diariamente.

RF03 — O sistema deve enviar notificações aos responsáveis.

RF04 — O sistema deve gerar relatórios para a coordenação.

RF05 — O sistema deve mostrar ranking de turmas organizadas.

---

## RNF — Requisitos Não Funcionais

RNF01 — O aplicativo deve ser fácil de usar.

RNF02 — O sistema deve funcionar rapidamente.

RNF03 — O sistema deve garantir segurança dos dados.

RNF04 — O aplicativo deve ser compatível com celulares.

RNF05 — O sistema deve estar disponível durante o horário escolar.

---

## RN — Regras de Negócio

RN01 — O uso da farda será registrado diariamente.

RN02 — Apenas a coordenação poderá alterar registros.

RN03 — Responsáveis receberão notificações.

RN04 — O sistema seguirá as normas da escola.
